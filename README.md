## How to run the app locally  
1. download and install Google App Engine SDK for Python
2. cd 
3. dev_appserver.py .
4. visit localhost:8080

## How to deploy 
1. download and install Google App Engine SDK for Python
2. cd 
3. appcfg.py update .
4. visit udacity-conference-center.appspot.com
